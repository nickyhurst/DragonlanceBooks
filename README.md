# Dragonlance Collection

A React app that catalogs Dragonlance novels, comics, and modules.

Features:

- React + Bootstrap UI
- Mobile card layout
- Desktop DataTable layout
- Infinite scroll on mobile
- Workbox service worker caching cover images
- Image modal viewer
- Search and filtering

## Development

```bash
npm install
npm run dev
```

## Build
```bash
npm run build
